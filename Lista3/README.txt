Kompilacja:
javac *.java

Uruchomienie:
1) Zadanie pierwsze znajduje sie w folderze Lista2 // uruchomienie z parametrem --type radix

2) 	java Main  -p randselect   	- wywolanie randomized selecta dla permutacji
	java Main  -r select 		- wywolanie selecta dla losowych danych
	java Main -p both 		- wywolanie obu algortymow dla takiej samej permutacji i porownanie statystyk

3) 	java BinSearch -t 		- i nastepnie podanie rozmiaru 30 i tablicy: 0,1,2,2,4,4,4,5,7,8,9,10,10,10,12,14,14,17,19,21,21,22,23,23,23,23,24,26,27,29
	java BinSearch -s 		- wywoluje BinarySearch w trybie zbierania statystyk

4) Zadanie czwarte znajduje sie w folderze Lista2
